# Media-generator
# 🎮 Claw Machine Content Factory

AI-powered dashboard to generate promotional images for claw machines in Laos, Myanmar, and neighboring countries.

## Features
- 🤖 Chat-style interface
- 🎨 4 AI models (Flux Kontext, Nano Banana Pro, Seedream 4, GPT-image-1)
- 📐 Social media size presets (Instagram, Facebook, Line OA)
- ☁️ Auto-upload to Cloudinary folder: `AI-image-gen`
- 🔗 One-click copy URL for Buffer/Line/Facebook
